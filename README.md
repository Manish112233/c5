## ğŸ§  Working Principle
1. The DC motor is driven using **PWM generated by Timer1**.
2. The motor encoder generates pulses proportional to the motor speed.
3. Encoder pulses are counted using **External Interrupt (INT0)**.
4. **Timer0** creates a precise 1-second measurement window.
5. RPM is calculated using the formula:

RPM = (Pulses per second Ã— 60) / Pulses Per Revolution (PPR)
6. The calculated RPM is displayed on the LCD in real time.
